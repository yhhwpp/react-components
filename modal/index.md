# Modal 弹出框


## API

| 参数             | 说明                    | 类型     | 默认值        |
|-----------------|-----------------------|----------|---------------|
| visible      | 对话框是否可见  | Boolean          | false           |
| title        | 标题           | React.Element    | 无           |
| footer       | 底部内容       |  Array [{text, onPress}]    | [] |
